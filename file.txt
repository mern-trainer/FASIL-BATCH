Git -> Version Control System
Github -> Host for Version Control System -> bitbucket, gitlab

-> Git Commands

git init -> Initialize a repository

git config --global user.name "username" -> Set username
git config --global user.email "email" -> Set email

git add . -> Stage files
git add <file_name_1> <file_name_2> -> Stage files

git commit -m "message" -> Commit files

git remote add <remote_name> <remote_url> -> Add remote repo

git remote -> List remotes
git config --get remote.<remote_name>.url -> Get remote url

git push <remote_name> <branch_name> -> Push files

-> Branch

git branch <branch_name> -> Create branch
git branch -> List branches

git switch <branch_name> -> Switch branch
git checkout <branch_name> -> Switch branch
